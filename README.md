# Prática 1 - Criando classes e objetos básicos

Este repositório contém as atividades da disciplina de Programação Modular ministrada pelo Professor Glender Brás.

## 🚀 Exercícios Desenvolvidos

### 1. Teorema de Pitágoras e Área do Triângulo
Implementação de uma classe chamada `TrianguloRetangulo` para calcular a hipotenusa e a área.
* **Conceitos aplicados**: Criação de classes, atributos e métodos com retorno.
* **Matemática**: Uso da biblioteca `Math` para raiz quadrada e potência.

### 2. IMC e Classificação Corporal
Implementação da classe `Pessoa` para cálculo de Índice de Massa Corporal.
* **Conceitos aplicados**: Encapsulamento (atributos `private`), métodos `get` e `set`, e construtores.
* **Lógica**: Classificação baseada na tabela oficial de IMC.

## 🛠️ Tecnologias Utilizadas
* Java (JDK Eclipse Temurin)
* Visual Studio Code
* Git para controle de versão (organizado em branches por exercício)
